# 0DTE $DGEN Trader

A fun educational clicker game where you can purchase 0DTE call and put options.

<img width="959" height="485" alt="image" src="https://github.com/user-attachments/assets/e8b542cd-2229-4a6f-a779-969281ad1005" />


<br>

Features

- Pricing and stock movement are calculated using Black Scholes and Geometric Brownian Motion to simulate real-life
- Mood-based advisor system that displays different expressions and dialogue depending on the players performance
- Leveling and XP system that correlates to player's stock market "wins"
